O FoodHub é um projeto acadêmico de IHC e UX inspirado em aplicativos como 99Food e iFood. A ideia foi criar um site simples de delivery para colocar em prática conceitos de usabilidade e experiência do usuário.

O projeto foi desenvolvido com HTML, CSS e JavaScript, e possui três etapas principais: escolher o restaurante, selecionar os produtos e finalizar o pagamento.

Durante a criação, o foco foi deixar a navegação fácil, organizada e intuitiva, garantindo uma boa experiência para quem utiliza a plataforma. É um projeto simples, mas que mostra na prática como funciona o desenvolvimento de interfaces e a aplicação de conceitos de IHC/UX.
